# FPL Squad Improver

A modern, data-driven web application that helps Fantasy Premier League (FPL) managers optimize their squads. Link your team, visualize your formation, and receive transfer suggestions based on real FPL data.

- **Smart Suggestions**: Receive data-driven recommendations to replace underperforming players within your budget.
